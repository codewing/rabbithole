## Game Description
Shooting game developed on 2D generated destructible terrain with strategy features. The game will make use of physics for the weapons, with the addition of pickups appearing on the game world. ## Setup

- Clone the repo with the submodules: `git clone --recurse-submodules`
- Build using CMake

## Working with this repository

- General/Reusable engine code can be found in `src/engine/`
- Game specific code is in `src/game`
